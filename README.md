# 四季について話す前に
CC0です。

同じく著作者人格権も不行使とします。

現代の法律との相性とかを一切考えず、会社の一生における自責上限と変則最低限の扱い方を発明しました。

名前を「四季」とします。

# 条件
・自責上限から運動という変数が取り除けている事

・A層の割合に比例して意思決定者を増減させる事

・ノルマが下がり切る位置を全員がB層で生きていける場所とすること

・自責上限のリセットタイミングを、従業員ごとに月火水木金の五分割すること

・A層の人間が自責上限が規定値に達した順にA層からB層へ移す事

・A層の定義を自責上限リセットタイミングでA層スタートの人間とすること

・変則最低限から割合規定を取り除く事。上限はA層7割B層3割、下限はA層1割B層9割

# 春→夏
新しい発明やらプロジェクトやらが始まってエイエイオーッて時期です

S層数名、A層1割B層9割から始めます。

B層がノってきたらA層を増やします。

同様に意思決定者も増やしてください。

A層を増やし続けてA層7割B層3割になったなら全体のノルマを上げる事。

変則最低限におけるノルマ低下条件にひっかかったならノルマを下げてください。

# 夏→秋
会社にお金貯まったねーって時期です。

大体の安定している企業はここにいます。

A層7割B層3割のままで良いでしょう。

景気や情勢に応じてノルマが下がったり上がったりする時期です。

# 秋→冬
景気や情勢が悪く、ノルマ未達が増えるタイミングです。

ノルマが下がり切る位置まで下がったらA層を減らしていってください。

同様に意思決定者も減らします。

意思決定者が持っていた決定権はその時点で残っているA層に渡す事とします。意思決定者を減らす事で権力をごく少数に集中させます。

ごく少数の人数が決定権を握る事で意思決定スピードが爆速で進みます。

ここでどうにもならなければ会社は消えます。

発明やら革新的なプロジェクトなどがあって、どうにかなった場合春に戻ります。

俺の予想では暇なB層からそれは出るんじゃないだろうかと思います。
